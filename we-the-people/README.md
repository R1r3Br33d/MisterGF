# We the People - Decentralized Blog Network\n\n> A community-owned, recommendation-driven blog platform combining MySpace customization, Reddit discovery, and Medium quality. No ads. No corporate influence. Pure peer-to-peer economy.\n\n## Vision\n\n**We the People** is a social blog network where:\n\n- 📝 **Individual blog pages** - Users own their space, customize it completely\n- 🔥 **Central hub** - Real-time trending topics organized by sectors\n- 📚 **Navigable archive** - Search/browse popular posts by topic, sector, and time\n- 💰 **Recommendation economy** - Users earn by being genuinely recommended by others\n- 🤝 **Community-funded** - No ads, funded entirely by voluntary patronage\n- 🔓 **Open & transparent** - Community controls governance and treasury\n\n## Core Features\n\n### 1. Individual Blog Pages\n\nEach user gets a customizable blog:\n\n```\nUser Profile + Blog Combined\n- Custom CSS styling (sandboxed)\n- Post archive\n- Follower system\n- Reader comments\n- Media galleries\n- About/Links section\n- Social presence\n```\n\n**Example User Blog:**\n```\n@alice's Blog\n━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━\n[Profile] Alice - Music Producer\n2.3k Followers | 156 Mentions | 94/100 Trust\n\n📌 Latest Posts:\n   • \"My New Album\" (2.3k views, 342 comments)\n   • \"Studio Setup Tour\" (8.9k views, 1.2k comments)\n   • \"Production Tips\" (23k views, 3.1k comments)\n\n🏷️ Sectors: #Music #Production\n🔗 Links: Spotify | SoundCloud\n💰 Support: Patronage | Tip Jar\n```\n\n### 2. NOT NULL,\n  password_hash VARCHAR(255) NOT NULL,\n  profile_image_url TEXT,\n  bio TEXT,\n  custom_css TEXT, -- Sandboxed\n  sectors TEXT[], -- Tags like 'music', 'art', 'tech'\n  followers_count INT DEFAULT 0,\n  following_count INT DEFAULT 0,\n  created_at TIMESTAMP DEFAULT NOW(),\n  updated_at TIMESTAMP DEFAULT NOW()\n);\n```\n\n### Blog Posts\n```sql\nCREATE TABLE blog_posts (\n  id UUID PRIMARY KEY,\n  author_id UUID REFERENCES users(id),\n  title VARCHAR(255) NOT NULL,\n  content TEXT NOT NULL,\n  sector VARCHAR(50), -- 'music', 'art', 'tech', etc.\n  tags TEXT[], -- Hashtags\n  views INT DEFAULT 0,\n  comments_count INT DEFAULT 0,\n  shares_count INT DEFAULT 0,\n  mention_count INT DEFAULT 0,\n  authenticity_score FLOAT DEFAULT 50,\n  recommendation_score FLOAT DEFAULT 0,\n  published BOOLEAN DEFAULT TRUE,\n  created_at TIMESTAMP DEFAULT NOW(),\n  updated_at TIMESTAMP DEFAULT NOW(),\n  published_at TIMESTAMP\n);\n```\n\n### Mentions\n```sql\nCREATE TABLE mentions (\n  id UUID PRIMARY KEY,\n  mentioner_id UUID REFERENCES users(id),\n  mentioned_user_id UUID REFERENCES users(id),\n  post_id UUID REFERENCES blog_posts(id),\n  context TEXT, -- What they said\n  authenticity_score FLOAT,\n  engagement_count INT DEFAULT 0,\n  reward_amount FLOAT DEFAULT 0,\n  status VARCHAR(20) DEFAULT 'pending', -- pending, verified, rejected\n  created_at TIMESTAMP DEFAULT NOW()\n);\n```\n\n### User Reputation\n```sql\nCREATE TABLE user_reputation (\n  user_id UUID PRIMARY KEY REFERENCES users(id),\n  mentions_received INT DEFAULT 0,\n  mentions_given INT DEFAULT 0,\n  engagement_score FLOAT DEFAULT 0,\n  trust_score FLOAT DEFAULT 50,\n  earned_value FLOAT DEFAULT 0,\n  referral_bonuses FLOAT DEFAULT 0,\n  updated_at TIMESTAMP DEFAULT NOW()\n);\n```\n\n### Trending Topics\n```sql\nCREATE TABLE trending_topics (\n  id UUID PRIMARY KEY,\n  name VARCHAR(100) NOT NULL,\n  sector VARCHAR(50),\n  mention_count INT DEFAULT 0,\n  post_count INT DEFAULT 0,\n  momentum FLOAT DEFAULT 0,\n  popularity_score FLOAT DEFAULT 0,\n  last_updated TIMESTAMP DEFAULT NOW()\n);\n```\n\n### Community Treasury\n```sql\nCREATE TABLE community_treasury (\n  id UUID PRIMARY KEY,\n  transaction_type VARCHAR(50), -- donation, patronage, expense\n  amount FLOAT,\n  description TEXT,\n  from_user_id UUID REFERENCES users(id),\n  public BOOLEAN DEFAULT TRUE,\n  created_at TIMESTAMP DEFAULT NOW()\n);\n```\n\n---\n\n## API Endpoints\n\n### User Management\n```\nPOST   /api/auth/register          # Create account\nPOST   /api/auth/login             # Login\nPOST   /api/auth/logout engagementWeight = 0.35;   // Mentions/posts = 35%\n  const momentumWeight = 0.15;     // Trending up/down = 15%\n  const recommendationWeight = 0.10; // Human mentions = 10%\n\n  const recencyScore = calculateRecency(topic.lastUpdated);\n  const engagementScore = (topic.mentions + topic.posts) / 100;\n  const momentumScore = calculateMomentum(topic);\n  const recommendationScore = topic.recommendations || 0;\n\n  return (\n    recencyScore * recencyWeight +\n    engagementScore * engagementWeight +\n    momentumScore * momentumWeight +\n    recommendationScore * recommendationWeight\n  );\n}\n```\n\n### Authenticity Scoring\n\n```typescript\nfunction scoreAuthenticity(mention: Mention): number {\n  let score = 0;\n\n  // Deep context (+30 points)\n  if (mention.context.length > 100) score += 30;\n\n  // Recommender history (+20 points)\n  const avgQuality = calculateHistoricalQuality(mention.mentioner);\n  score += avgQuality * 20;\n\n  // Engagement (+25 points)\n  const engagementRatio = mention.engagement / mention.mentioner.followers;\n  score += Math.min(engagementRatio * 100, 25);\n\n  // Follower relationship (+15 points)\n  if (mention.mentioner.follows(mention.mentioned)) score += 15;\n\n  // Natural timing (+10 points)\n  if (isNaturalCadence(mention.mentioner)) score += 10;\n\n  return Math.min(score, 100);\n}\n```\n\n---\n\n## Community Governance\n\n### Decision-Making\n- All major changes voted on by community\n- Public proposal system\n- Transparent voting records\n- Execute decisions via smart
